docker——制作镜像（image）供比赛平台pull

  1. 新建镜像库，获得镜像库公网地址

    • 阿里云

      https://cr.console.aliyun.com/

    • 公网地址

      registry.cn-shanghai.aliyuncs.com/weila/test_for_tianchi_submit

  2. 制作镜像推送至镜像库

    • 登录账号：docker login –username=t_1489244214687_019 registry.cn-shanghai.aliyuncs.com

    • 创建镜像：docker build -t registry.cn-shanghai.aliyuncs.com/weila/test_for_tianchi_submit:1.0 .

    • 使用”docker tag”命令重命名镜像，并将它通过专有网络地址推送至Registry。

      查看镜像id

      docker images

      docker tag 07175153e661 registry.cn-shanghai.aliyuncs.com/weila/test_for_tianchi_submit:1.0

    • 提交镜像到云端镜像库

      docker push registry.cn-shanghai.aliyuncs.com/weila/test_for_tianchi_submit:1.0

  3. 比赛提交结果

    registry.cn-shanghai.aliyuncs.com/weila/test_for_tianchi_submit:1.0

    私有库要填写用户名密码
