结论：

  1. 有超长文本（5万字符以上），但这部分数据很少，所以可以通过截取的方法来进行处理。
  2. 中位数是676个字符，平均数是907个字符；但这是未处理过的文本长度，处理后会进一步缩减；（几百个字符实在太多了，BERT最多才能处理512个字符的长度，所以要经过去停留词、去高频词等方法来减少字符）

y:赛题数据的类别分布

python

  • 在数据集中标签的对应的关系：{‘科技’: 0, ‘股票’: 1, ‘体育’: 2, ‘娱乐’: 3, ‘时政’: 4, ‘社会’: 5, ‘教育’: 6, ‘财经’: 7, ‘家居’: 8, ‘游戏’: 9, ‘房产’: 10, ‘时尚’: 11, ‘彩票’: 12, ‘星座’: 13}
  • 数据集分布较为不均匀，第0、1、2类占了大部分，这就很容易对这三类的数据过拟合，而其他类会欠拟合。

结论

  • x：每个新闻包含的字符个数平均为907个，文本长度2-57921不等，有少数过长的，需要进行字符缩减，甚至截断等处理；
  • y：类别分布不均匀，科技类新闻样本量接近4w，星座类新闻样本量不到1k，可能会影响模型精度；
  • 字符：赛题总共包括6869种字符，可以推测出一些字符为标点符号，一些出现次数过少的字符也可以考虑去掉，需要想办法寻找这个阈值
